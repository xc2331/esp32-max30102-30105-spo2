# Contributing / 贡献指南

English | [中文](#中文)

Thank you for your interest in contributing!

欢迎贡献！

## How to Contribute / 如何贡献

1. Fork this repository / Fork 本仓库
2. Create your feature branch / 创建功能分支: `git checkout -b feature/your-feature`
3. Commit your changes / 提交更改: `git commit -m 'Add some feature'`
4. Push to the branch / 推送到分支: `git push origin feature/your-feature`
5. Create a Pull Request / 创建 Pull Request

## Reporting Issues / 报告问题

- Use GitHub Issues to report bugs or suggest features
- 使用 GitHub Issues 报告问题或建议功能

## Code Style / 代码风格

- Follow existing code conventions / 遵循现有代码约定
- Keep code clean and readable / 保持代码简洁易读
